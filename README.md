# ReadGEXF
A simple GEXF file reader by C++

## Version 0.1
This version can read gexf file generated by Gephi.

## Build
g++ main.cpp tinyxml2.lib -o readgexf.exe

or

Use the Visual Studio project (Based on VS2015 CTP)

## Run
in.gexf is the input file.
